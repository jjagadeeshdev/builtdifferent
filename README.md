# 🚗 Built Different

A modern car showcase website built using HTML and CSS.

## 🔥 Features
- Stylish UI with background effects
- Navigation bar with multiple sections
- Detailed car descriptions
- Image-based layout design
- Social media footer links

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📌 Project Highlights
- Clean layout and structured sections
- Use of flexbox for alignment
- Creative design with blur effects and transparency
- Beginner-friendly frontend project

## 🎯 Purpose
This project was created to practice and improve my frontend development skills, focusing on layout design and styling.
